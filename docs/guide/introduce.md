# Opuntia UI

Vue 3 UI 组件库起始模板，提供文档支持

## 使用环境

Opuntia UI 可以在支持 ES2018 的浏览器上运行。 如果您确实需要支持旧版本的浏览器，请自行添加 [Babel](https://babeljs.io/) 和相应的 Polyfill。

由于 Vue 3 不再支持 IE11，Opuntia UI 也不再支持 IE 浏览器。
